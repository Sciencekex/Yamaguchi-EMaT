# Yamaguchi‑EMaT 確率的出題練習システム

山口大学大学院創成科学研究科（工学系）修士課程入試の応用数学試験対策として、**工学系数学統一試験（EMaT）** の過去問を活用したランダム出題型の学習支援ツールです。

## ウェブサイト

<https://sciencekex.github.io/Yamaguchi-EMaT/>

## 機能概要

山口大学大学院入試の数学問題が**EMaT に準拠して作成される**ことに基づき、2003 年度～2025 年度の EMaT 過去問をデータベース化し、ランダムに問題を抽出して練習できるシステムです。

主な機能は以下の通りです。

- 微分積分、線形代数、確率・統計の各分野からランダム出題
- 隙間時間やリラックス時に手軽に基礎問題を演習可能
- 山口大学入試の出題形式・難易度に完全に対応
- 過去問の反復学習により、入試の確実性を高める

## 対象

山口大学創成科学研究科（工学系）修士課程を志望する受験生

## 備考

本システムは、山口大学入試の**応用数学試験が EMaT 過去問を基に出題される**方針に沿って開発されています。

「確実性のある学習」をコンセプトに、負担なく継続できる基礎演習を目的としています。
